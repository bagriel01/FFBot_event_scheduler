# FFBot_event_scheduler
Event Scheduler Bot written in Python for Telegram. 


Current features are:

/FFPost 
Redirects a post from a group to a channel after being approved by an approver

/FFNewPost
Guides the process of post creation and sends the post to a channel after being aproved by an approver

Future Features:

/ThisMonth
Shows the events published at the channel happening at the current month
